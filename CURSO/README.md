# CURSO

Contenido del programa "Liderar con Consciencia" (7 módulos).

## Filosofía

Viaje de autoconocimiento y desarrollo de liderazgo consciente, desde las bases neurobiológicas hasta un modelo ético de liderazgo. Progresión en círculos concéntricos: del yo interior → relación con otros → acción con propósito.

## Tono

- Rigor científico accesible (neurociencia, psicología)
- Invitación a la reflexión, no imposición
- Equilibrio teoría-práctica
- Respeto por la experiencia del participante
