## Validation Loop - Sección de Curso

### Level 1: Estructura
- [ ] Tiene objetivo de aprendizaje claro
- [ ] Incluye fundamentación científica
- [ ] Contiene ejercicio práctico
- [ ] Tiene resumen de ideas clave
- [ ] Extensión entre 500-1500 palabras

### Level 2: Rigor científico
- [ ] Afirmaciones clave tienen referencia
- [ ] Referencias son de fuentes verificables
- [ ] No hay afirmaciones inventadas

### Level 3: Pedagogía
- [ ] Progresión clara: concepto → evidencia → aplicación → práctica
- [ ] Lenguaje accesible para ejecutivos
- [ ] Al menos un ejemplo aplicado al liderazgo
- [ ] Ejercicio con instrucciones claras y reflexión

### Level 4: Coherencia
- [ ] Conecta con módulo/sección anterior
- [ ] Prepara para módulo/sección siguiente
- [ ] Consistente con el tono del curso
