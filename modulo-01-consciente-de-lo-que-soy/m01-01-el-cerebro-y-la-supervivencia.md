# El cerebro como máquina de supervivencia

#teoria #neurociencia

El cerebro se ha desarrollado evolutivamente para ayudarnos a sobrevivir. Cada estructura y función cerebral contribuye a anticipar y adaptarse a los cambios. No se trata solo de percibir el presente, sino de proyectar y prepararnos para el futuro.

## El modelo del mundo

#teoria #modelo-mental

El cerebro crea un "modelo del mundo" para anticipar eventos futuros. La evolución ha favorecido este sistema por su capacidad de aumentar las posibilidades de supervivencia.

Las redes neuronales, la memoria, los sentidos, la corteza asociativa y las emociones trabajan en conjunto para generar este "teatro mental" que nos permite navegar por el mundo e interactuar con él.

```
Supervivencia
     ↑
┌────┴────┐
│ Modelo  │
│   del   │
│  mundo  │
└────┬────┘
     ↑
┌────┼────┬────────┐
│    │    │        │
Memoria  Sentidos  Emociones
```
