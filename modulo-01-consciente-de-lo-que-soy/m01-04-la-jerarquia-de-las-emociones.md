# La jerarquía de las emociones

#teoria #emociones

Las emociones juegan un papel crucial en la forma en que recordamos y aprendemos de nuestras experiencias. Cuando vivimos algo emocionalmente intenso —ya sea positivo o negativo—, el cerebro "marca" ese evento como importante.

## El mecanismo evolutivo

#teoria #neurociencia

Esto se debe a que las emociones desencadenan la liberación de neuromoduladores, como la dopamina. Este sistema tiene una función evolutiva clara:

- **Eventos positivos** → dopamina → "repite esto"
- **Eventos negativos** → cortisol/adrenalina → "evita esto"

El problema es que este mecanismo no distingue entre amenazas reales (un depredador) y amenazas sociales (una crítica en una reunión). Para el cerebro primitivo, ambas activan respuestas similares.
