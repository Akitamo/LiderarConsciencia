# El círculo de la atención: entrenando mindfulness

#teoria #practica-mindfulness

El entrenamiento de mindfulness sigue un patrón cíclico que fortalece progresivamente la atención:

```
    ┌─────────────────────┐
    │  1. Concentración   │
    │   en el objeto      │
    └─────────┬───────────┘
              │
              ▼
    ┌─────────────────────┐
    │  2. La mente        │
    │   se distrae        │
    └─────────┬───────────┘
              │
              ▼
    ┌─────────────────────┐
    │  3. Darse cuenta    │
    │   de la distracción │
    └─────────┬───────────┘
              │
              ▼
    ┌─────────────────────┐
    │  4. Soltar el       │
    │   objeto distractor │
    └─────────┬───────────┘
              │
              ▼
    ┌─────────────────────┐
    │  5. Volver al       │◄────┐
    │   objeto de atención│     │
    └─────────────────────┘     │
              │                 │
              └─────────────────┘
```

*Fuente: Malinowski, P. (2013). Neural mechanisms of attentional control in mindfulness meditation.*

## El triángulo de la atención

#teoria #triangulo-atencion

Los tres "afluentes" de la consciencia que observamos en la práctica:

- **Pensamientos**: El flujo mental de ideas, juicios, narrativas
- **Sensaciones**: Información del cuerpo (interocepción)
- **Emociones**: Estados afectivos que colorean la experiencia
