# Prime Curso

Carga el contexto completo del curso para trabajar con el contenido.

## Acciones

1. **Leer contexto del curso**:
   - CURSO/README.md (filosofía y tono)
   - CURSO/CLAUDE.md (instrucciones de modificación)

2. **Leer estructura del curso**:
   - CURSO/00-Indice-General.md (si existe)
   - Estructura de módulos disponibles

3. **Cargar ejemplos de referencia**:
   - _ai/examples/seccion-curso-modelo.md

4. **Confirmar contexto cargado**:
   Responder con resumen breve:
   - Módulos identificados
   - Estado general del curso
   - Listo para recibir instrucciones

## Uso

```
/prime-curso
```

## Notas

- Ejecutar al inicio de sesiones de trabajo con contenido del curso
- No modifica ningún archivo, solo carga contexto
