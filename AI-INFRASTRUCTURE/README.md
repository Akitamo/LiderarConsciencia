# AI Infrastructure

Recursos para trabajar con IA en el desarrollo del curso.

## Carpetas

- **prompts/** → Prompts reutilizables
- **skills/** → Capacidades modulares (por desarrollar)
- **agents/** → Agentes especializados (por desarrollar)
- **workflows/** → Flujos de trabajo (por desarrollar)
- **history/** → Registro de sesiones (por desarrollar)

## Prompts disponibles

| Prompt | Descripción |
|--------|-------------|
| `Prompt asociar imagen placeholder.md` | Asociar imágenes a contenido del curso |
| `Prompts extracción PPTs.md` | Extraer contenido de presentaciones PowerPoint |
| `Prompt reestructuracion pedagogica modulo.md` | Reestructurar ficheros de un módulo con visión global + ejecución fichero a fichero |
