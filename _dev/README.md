## Estructura por proyecto

- `README.md`: Visión y contexto
- `PLAN.md`: Plan general (funcionalidades, estado, orden)
- `docs/`: ADRs y decisiones técnicas
- `specs/`: Especificación por funcionalidad
- `examples/`: Patrones y ejemplos
- `prompts/`: Prompts específicos del proyecto
- `scripts/`: Código
- `config/`: Configuración
- `work/`: Checkpoints intermedios
- `output/`: Resultados
